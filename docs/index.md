# Otherworld [Dungeons & Dragons]

> **Minecraft 1.20.1 | Forge 47.4.13 | ~392 Mods | CurseForge Pack ID: 1418133 (v5)**

Welcome to the complete guide for the **Otherworld [Dungeons & Dragons]** modpack. This wiki explains every custom datapack, crafting recipe, configurable option, and progression system in the pack.

---

## What is Otherworld?

Otherworld is a heavily customized Minecraft modpack that transforms the game into a D&D-inspired action RPG. It features:

- **Diablo-style loot** with 6 rarity tiers and 80+ random affixes (Apotheosis)
- **Distance-based mob scaling** that gets harder the further you explore (Autoleveling)
- **Action RPG combat** with combo chains, weapon animations, and hitboxes (Better Combat)
- **58+ bounty integrations** for quest-based gameplay (Extra Bounties)
- **Wave-based mob arenas** in 3 difficulty sizes (Gateways)
- **Temperature and thirst** survival mechanics (Legendary Survival)
- **Cross-mod compatibility** ensuring 392+ mods work together seamlessly

## How the Pack is Organized

All custom datapacks live in:

```
minecraft/config/paxi/datapacks/
```

**Paxi** is a mod that automatically loads datapacks from this folder â€” no need to manually install them into each world save. Any changes you make here apply to all worlds.

There are **7 custom Otherworld datapacks** (folders) and **14 additional datapacks** (zip files).

### Key Config Locations

| What | Where |
|------|-------|
| Datapack files | `minecraft/config/paxi/datapacks/` |
| Mod configs (.toml, .cfg) | `minecraft/config/` |
| Resource packs | `minecraft/resourcepacks/` |
| Mod JARs | `minecraft/mods/` |
